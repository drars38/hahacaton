# Быстрый старт

## 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

## 2. Применение миграций

```bash
python manage.py migrate
```

## 3. Запуск сервера

```bash
python manage.py runserver
```

## 4. Открыть в браузере

Перейдите по адресу: http://127.0.0.1:8000/

## 5. Загрузка больших датасетов (оптимизированный способ)

Для загрузки больших CSV файлов из папки используйте команду:

```bash
python manage.py load_datasets "C:\Users\arsenii\Desktop\hacaton\ТОНАЛЬНОСТЬ"
```

Или с указанием размера батча:

```bash
python manage.py load_datasets "C:\Users\arsenii\Desktop\hacaton\ТОНАЛЬНОСТЬ" --batch-size 2000
```

Эта команда:
- Автоматически найдет все CSV файлы в указанной папке
- Обработает их с оптимизацией (батчевая обработка)
- Покажет прогресс загрузки
- Пропустит уже загруженные файлы

## 6. Тестирование

1. Загрузите файл `example_data.csv` через интерфейс
2. Дождитесь обработки
3. Просмотрите результаты в таблице
4. Попробуйте фильтры и поиск
5. Экспортируйте результаты в CSV
6. Загрузите валидационный файл для проверки метрик

## Формат валидационного файла

Создайте CSV файл с колонкой `sentiment` (0, 1 или 2):

```csv
text,sentiment
"Это отличный продукт!",2
"Не понравилось",0
"Нормально",1
```
